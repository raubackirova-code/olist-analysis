Анализ продаж интернет-магазина Olist
О проекте

В рамках проекта проведён анализ транзакционных данных бразильского маркетплейса Olist с целью выявления:
 - структуры выручки,
 - ключевых клиентских сегментов,
 - наиболее прибыльных категорий товаров,
 - вклада продавцов в общий оборот компании.

Проект демонстрирует навыки работы с реляционными данными и перевод аналитических выводов в бизнес-рекомендации.

Бизнес-цели:

 - Определить целевую аудиторию и её вклад в выручку.
 - Выявить наиболее продаваемые категории товаров для оптимизации хранения и логистики.
 - Оценить вклад продавцов и определить стратегию работы с ними.
 - Проанализировать структуру выручки в целом.

Использованные данные

В анализе использованы таблицы:
 - customers
 - orders
 - order_items
 - products

Объединение таблиц выполнено по ключам:
 - customer_id
 - order_id
 - product_id

Выручка рассчитана как:
 revenue = price + freight_value

Использованные инструменты
 - Python
 - Pandas
 - NumPy
 - Matplotlib
 - Jupyter Notebook

Применённые методы:
 - очистка и проверка данных
 - объединение таблиц (merge)
 - группировки и агрегации (groupby, agg)
 - сегментация клиентов
 - анализ вклада категорий
 - визуализация данных

Основные результаты

Клиентская база
Клиенты сегментированы на:
 - Gold
 - Silver
 - Regular

Результаты:
 - Gold + Silver формируют 53,5% общей выручки.
 - Regular (80% клиентской базы) обеспечивает 46,5% выручки.

Выводы :
Компания имеет устойчивую массовую аудиторию и выраженный сегмент лояльных клиентов.
Категории товаров-лидеров по выручке:
 - health_beauty
 - watches_gifts
Также значительный вклад вносят:
 - bed_bath_table
 - computers_accessories
Есть потенциал расширения ассортимента в лидирующих категориях и необходимость учитывать сезонность при планировании запасов.

Продавцы
Анализ показал концентрацию выручки среди части продавцов.

Вывод:
Целесообразно внедрить систему мотивации для топ-продавцов.
Разработать программу развития для продавцов среднего сегмента.

Бизнес-рекомендации
 - Внедрение программ лояльности для Gold и Silver клиентов.
 - Работа с Regular сегментом по увеличению среднего чека.
 - Расширение ассортимента в ключевых категориях.
 - Оптимизация логистики с учётом сезонного спроса.
 - Мотивационные программы для продавцов.

Дополнительно анализ выполнен в SQL (DBeaver).
